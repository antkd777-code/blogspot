경쟁 브랜드 분말 제품 대신, **분말 vs 영양제 비교 구도**로 루토시아잔틴 전환.

| 키워드 | 전략적 의미 |

| 빌베리분말 | 메인 검색 유입 키워드 |
| 빌베리가루 | 유사 표현, 함께 배치 |
| 블루베리분말 | 경쟁 키워드 (빌베리 vs 블루베리 비교 훅) |

## 6. 트렌드 분석

- "빌베리분말" 검색 트렌드: 건강 관심층의 자가 건강 관리 수요 증가
- 분말 형태 건강식품 → 편의성 부족으로 캡슐/정제 전환 니즈 높음
- 블루베리 대비 빌베리 인지도 상승 중 (안토시아닌 함량 비교 콘텐츠 증가)
- 4060 타겟층의 눈건강 관심 → 빌베리 성분 탐색 → 고함량 복합제 전환 흐름
