## 키워드 배치 검수

| 키워드 | 목표 | 실제 배치 위치 | 판정 |
|--------|------|---------------|------|
| 시력에 좋은 음식 | 3~4회 | 제목, 인트로, 4위 소제목, 후기 | ✅ 4회 |
| 눈건강 | 2~3회 | 1위 소제목, 2위 본문, 전환섹션 | ✅ 3회 |
| 눈에 좋은 음식 | 1~2회 | 인트로, 3위 본문 | ✅ 2회 |
| 빌베리 | 1~2회 | 3위 소제목·본문, 전환섹션 | ✅ 2회 |
| 차즈기 | 1~2회 | 4위 소제목·본문, 전환섹션 | ✅ 2회 |
| 헤마토코쿠스 | 1회 | 5위 소제목·본문, 전환섹션 | ✅ 2회 |
| 루토시아잔틴 | 2~3회 | 전환섹션, 후기, 제목 | ✅ 3회 |

전체 키워드 노출 합산: 18회 ✅

---

## SEO 개선 제안

1. 제목에 "1위" 숫자 노출로 클릭률(CTR) 강화 → 현재 제목 유지 OK
2. 인트로 첫 문장에 메인 키워드 정확히 포함 → ✅ 확인
3. 각 섹션 소제목에 서브 키워드 자연 배치 → ✅ 확인
4. 결론(후기) 섹션에 메인 키워드 재등장 → ✅ "시력에 좋은 음식 찾다가..." 확인

---

## 메타 디스크립션

시력에 좋은 음식 1위부터 5위까지 랭킹 정리해봤어요.
당근, 비타민A, 빌베리, 차즈기, 헤마토코쿠스까지!
특히 3~5위 성분 함께 챙기는 법도 공유해요.

(89자 — 네이버 블로그 메타 80~100자 권장 범위 내 ✅)
